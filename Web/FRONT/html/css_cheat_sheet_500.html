Este documento tiene como objetivo brindar información referente al uso de CSS en una página web. Con este fin, se explicarán y ejemplificarán ciertas propiedades.

Borders

Se utilizan para estilizar los bordes de una página y existen distintas maneras de hacerlo. A continuación se explican algunas de estas:

Border

Este es el identificador general para estilizar el borde completo. Los principales valores que le pueden ser asignados son: ancho de línea, estilo de línea y color (existen distintas etiquetas para esto). Lo anterior se puede ejemplificar visualmente en esta sección.

Border-left, border-right, border-top, border-bottom

Estos sirven para estilizar únicamente un lado definido del borde dentro de un contenedor, al igual que la propiedad general de borde, se aceptan valores de ancho, estilo de línea y color. Border-left, como su nombre lo indica, estiliza el borde izquierdo. De manera similar, border-right, border-top y border-bottom estilizan secciones específicas del borde (esto se ejemplifica con bordes de distintos colores).

Background-image

Esta propiedad contribuye a insertar una o más imágenes en el fondo de la sección en cuestión (como se muestra aquí).

Color

Esta propiedad sirve para cambiar el color del texto en una página. Es posible hacer esto de distintas maneras: se puede utilizar el nombre del color (ej. red), utilizar el número hexadecimal del color (#ff00ff) o mediante la propiedad rgb(rojo, verde, azul) editando los valores numéricos de cada uno de los tres colores.

Fonts

Los fonts nos ayudan a editar el estilo del texto. Algunas de las características editables son: tamaño del texto, tipo de texto (italico, bold, etc), tipo de letra (arial, helvetica, etc), entre otras.

Font

La propiedad font, nos ayuda a estilizar todo lo relacionado con la fuente de letra. Font es una manera de asignar valores generales a dicha fuente como se muestra en esta sección.

Font-family

Esta propiedad especifica una lista de familias de fuentes por usar. En este caso, sí se cuenta con una prioridad; se separan con comas las familias para indicar que son alternativas. Con esto, si no existe un caracter en cierta familia de font, es posible utilizar otro font de la lista. La implementación de una sola familia de font se puede observar aquí.

Font-style, font-weight

En este caso, font-style sirve para definir si un font debe ser estilizado con italica, normal u oblicuo. Para esta propiedad, normal no agrega ningún estilo, italico utiliza menos espacios horizontales entre letras, y oblicuo implica inclinación para letras (italica y oblicuo se ven muy parecidas). Dentro de esta sección estamos ejemplificando oblicuo.

Similarmente, font-weight nos ayuda a especificar si un font debe ser estilizado con bold, normal o light (suave). En este ejemplo, estamos utilizando bold.

Ambas propiedades pueden ayudarnos a darle significado a ciertas secciones de texto.

Width, height

Las propiedades width y height nos ayudan a definir las dimensiones de un área de contenido; como sus nombres lo indican, height sirve para especificar el alto del cuadro, mientras que width sirve para el ancho. Se definen valores numéricos o porcentuales para definir esto (nos encontramos en un ejemplo de las propiedades).

En este ejemplo, utilizamos "auto" para que la caja se ajuste al texto.

List-style

Como su nombre lo indica, esta propiedad nos sirve para dar estilos a listas. Nos es posible definir desde la parte exterior de la lista, hasta aspectos dentro de la estructura de la lista. En este caso, se posicionaron los elementos "adentro" de las lista y dichos elementos tienen una imagen en lugar de puntos.
