Párrafos (p)

En cuanto a los párrafos, son definidos por el tag "p" nos sirve para insertar párrafos (se inserta una caja de texto). Este texto es un ejemplo claro del uso del tag.

Links (a)

Se usa tag "a" para marcas de hyperlink acompañado de href (para decirle a dónde ir). Esto nos ayuda a referenciar un lugar específico (puede ser otro archivo html o una página)

Va a la página del link

El atributo "Target" nos ayuda a definir si se va a cargar todo en la misma página o si se quiere ir a una nueva pestaña

Divisiones (div)

El tag "div" sirve de contendor que nos ayuda a estructurar y dar estilo posteriormente. Básicamente, nos sirve para identificar grupos de contenido y poder estilizarlos de manera simple.

En este documento html, se ejemplifica el uso del div por cada tag explicado (aunque visualmente no se vea, estructuralmente está presente).

Strong (strong) y bold (b)

El tag definido por la palabra "strong" nos sirve para dar mucha importancia a cierto pedazo de texto además de que se puede incluir dentro de una línea. Un ejemplo de esto es lo siguiente: importante

Podría considerarse que "strong" es un tag semántico ya que sí tiene como objetivo recalcar la importancia de algo, mientras que existe otro tag cuyo único propósito es el estilo (en este caso poner las letras en bold): "b".

Podemos ver cómo el tag para bold "b" tiene un efecto igual al de strong, con la diferencia de que se utiliza más para dar estilo (o por lo menos eso se intenta): importante

Italica (i) y (em)

En el caso de estos tags, tenemos un caso similar al de "strong" y "b". Siendo el tag "em" utilizado mayormente para dar énfasis, por ejemplo: importante. Por otro lado, el tag "i" se utiliza para dar cierto tono, por ejemplo: importante. A pesar de que ambos tags se ven iguales, nuevamente lo que varía es el significado que tienen.

Subrayado (u)

El tag "u" nos es de gran utilidad para subrayar cierta sección de texto. Por lo general es usada para marcar palabras mal escritas. Su implementación se ve de la siguiente manera: subraia

Tachado (strike)

El tag "strike" nos sirve para tachar una sección de texto como se muestra: no sirve. Este elemento ya no es usado en HTML5 (de hecho se marca un error al utilizarlo, a pesar de que sigue funcionando) por lo que en su lugar se pueden usar los tags "del" del o "s" s.

Texto pequeño (small)

El tag "small" nos sirve para hacer el texto más pequeño. Puede ser utilizado para comentarios y se ve así: hola en pequeño

Tags Semánticos

Los tags semánticos agregan significado. Es decir, cuando usamos cierto tag semántico, nos sirve para identificar el significado que le queremos dar a cierto segmento de estructura dentro del HTML.

Dentro del header

Esencialmente un div, pero con significado semántico (nos dice que es un div para header específicamente)

Este tag no se puede ejemplificar visualmente, sin embargo, actualmente nos encontramos dentro de un header.

Dentro del span

Contenedor para fraseo. Es muy parecido a div, pero en vez de funcionar en bloques, funciona por línea. Nos ayuda a estilizar una línea.

Este tag no se puede ejemplificar visualmente, sin embargo, actualmente nos encontramos dentro de un span.

Dentro de section

Este tag, como su nombre lo indica, nos sirve para definir una sección en un documento. Puede ayudarnos a identificar un grupo de contenidos sobre cierto tema.

Este tag no se puede ejemplificar visualmente, sin embargo, actualmente nos encontramos dentro del section.

Dentro del article

Contenedor para composición en un documento, página, etc, que puede ser usado de manera independiente.

Este tag no se puede ejemplificar visualmente, sin embargo, actualmente nos encontramos dentro del section.

Dentro de aside

El tag "aside" sirve para insertar contenido aparte del que se incluye en el resto de la estructura del html. Es útil para contenidos tales como barras laterales o elementos insertados. Al igual que otros tags que fungen como contenedores, en este caso no es posible observar en la página un ejemplo de "aside", pero estamos dentro de este.

Dentro de footer

El tag "footer" es utilizado para incluir contenido en un pie de página. Es común incluir dentro de este tag información de contacto, información de copyrights, documentos relacionados, entre otros.

Este tag tampoco se puede visualizar en la página, sin embargo estamos dentro de un footer.

Dentro de nav

El tag de "nav" nos sirve para insertar bloques con links de navegación. Un ejemplo de esto podría ser la tabla de contenidos en una página.

Listas

En general, los tags relacionados con las listas nos ayudan a mantener la estructura ya sea de una lista ordenada y la de una lista sin orden. Cada elemento de una lista se define con el tag "li" (sin importar que sea ordenada o sin orden, se verá un ejemplo más adelante).

Lista sin orden (ul)

El tag "ul" (junto con "li" para cada elemento), nos ayuda a enlistar elementos sin importar el orden (se ponen "puntos"). Podemos ver el siguiente ejemplo en donde "ul" tiene distintos "li" adentro:

  • Manzana
  • Pera

Lista ordenada (ol)

El tag "ol" (junto con "li" para cada elemento de la lista), nos ayuda a enlistar elementos dando importancia al orden (se ponen "números") como en el siguiente ejemplo:

  1. Primaria
  2. Secundaria

Tablas (table)

Nos ayudan a organizar la información en filas y columnas (tr = table row, td = celda). En este caso, se utilizan tags diferentes: "table" para crear en términos generales la tabla, "tr" para las filas de la tabla, "th" para el encabezado de las columnas, "td" para definir una celda de datos. El uso de estos elementos se puede ejemplificar con lo siguiente:

h1 h2 h3
r11 r12 r13
r21 r22 r23
r31 r32 r33

Además, en este caso se tiene la estructura separada con los tags "thead" para el encabezado y "tbody" para lo demás.
